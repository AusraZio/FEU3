CSS rasymo budai

inline - linijinis

linijinis stilius yra, kai elementui kuri stilizuojame suteikiame style atributa ir jo viduje aprasome stilius. Neturi perpanaudojimo galimybes ir yra skirtas tiktaias testavimui.

internela- vidinis

vidinis - vidinis stylius yra, kai HTNL failo viduje, head tag'o dalyje pasirasome style tag'a ir tame style tag'e rasom CSS koda. Turi siek tiek perpanaudojimo galimybiu (nes turiselektorius) ir dazniausiai naudojamas testavimui.

external - isornis

isornis stilius yra rasomas atskiras CSS faile ir yra prijungiamas prie HTML failo naudojant link tag'a. Turi visas reikalingas/norimas perpanaudojimo galimybes, yra placiausiai naudojamas ir galutinio projekto variante, turetu likti tik sis stiliaus rasymo budas

Basic Selectors

Element / Tag

selektinami visi pagal nurodyta elementa tag'as.
Pvz.: h2 -> parinks visus h2 elementus esancius HTML'e.

Class

Klase sukuriame kokiu tik nori logisku ir trumpu pavadinimu. CSS'e tam, kad nurodytume, kad kurimae klase, sugalvoto pavadinimo pradzioje parsome taska (.).HTML'e norint priskirti klase elementui, tam elementui suteikiam atributa "class" ir jam priskiriame sugalvota klases pavadinima be tasko.
Pvz.: CSS'e: .klasesVardas; HTML'e: < class="klasesvardas">

ID

ID sukuriame kokiu tik norime logisku ir trumpu pavadinimu. CSS'e tam, kad nurodytume, jog kuriame ID - sugalvoto pavadinimas pradzioje parasome groteles.HTML'e norint priskirti ID elementui, tam elementui suteikiame atributa "id" ir jam priskiriame sugalvota ID pavadinima be groteliu.
Pvz.: CSS'e: #idPavadinimas; HTML'e: <p id="idPavadinimas">
ID turi vienas su vienu santyki. Vienas elementas gali tureti TIK vienam ID. ir vienas ID turetu buti priskirtas Tik vienam elementui.

Colors

Names

Splavos gali tureti vardus. VArdiniu spalvu yra 140. Norit priskirti elementui spalva, priskiri tos spalvos varda.
Pvz.: red; blue; green; lightgrey; dargreen; ...

RBG

RBG - red green blue, kiekvienos is siu spalvos kiekio galima ideti nuo 0 iki 255 ir sitaip isgauti bet kuria splva.
Pvz.: rgb(0, 0, 0) - juoda. rbg(255,0, 0) - raudona;..
RGBA- red green blue alpha. Alpha nurodo splavos matomuma ir jis gali buti nuo 0 iki 1 tukstantaja paklaida.
Pvz.: rgba(0, 255, 0, 0) - nematoma(green/zalia); rgba(0, 255, 0, 1) - pilnai matoma(zalia;)...

hex

Hex - hexadecimal - sesioliktainis. Kiekvienos is spalvos (RGB) galima nurodyti nuo 00 iki ff ir sitaip isgauti bet kuria spalva.
Skaiciavimas sesioliktaineje sistemoje: 00 01 02 03 04 05 06 07 08 09 0a 0b 0c 0d 0e 0f 10 11 12 13 ... 19 1a 1b... 1f 20 21 22 ... 9f a0 a1 a2... a9 aa ab ac ad ae af b0 .... ff
Pvz.: #000000 - juoda; #ff0000 - raudona ...
Trumpiniai: #000 - juoda; #f00 - raudona; ...
spalvos matomumas su Hex:gale prisideda dar su sesioliktainiai simboliai, kur 00 - permatomas, ff - pilnai matomas.
Pvz.: #0f00 - nematoma (zalia); #00ff00 - pilnai matoma (zalia) spalva; ..
trumpiniai:#0f00 - nematoma (zalia); #0f0f - pilnai matoma(zalia); ...

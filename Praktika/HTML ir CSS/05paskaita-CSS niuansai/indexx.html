Paveldejimas

Paveldeti stilius gali tik vaikinis elementais tevinio elemento.
Paveldimi stiliai yra, tie kurie keicia elemento izvaizda, ne busena. Pvz.: color, font-size, font-family...
Nepaveldimi stiliai yra tie, kurie keicia elemento busena. Pvz.: width, height, display,border, position...
Net paveldeti stiliai, gali nebuti atvaizduoti del stiliu prioritetu.

Numatytieji stiliai

Numatytieji stiliai yra tag'ams automatiskai narsykles suteikti stiliai. Tarp skirtingu narsykliu, numatytieji stiliai gali minimalistiskai keistis ir sitaip sugadinti svetaines vaizda skirtingu narsykliu vartotojams.
Tam, kad nebutu problemu kuriant dizaina skirtingoms narsyklesms, yra naudojami Rest srba Normalize CSS'ai.
Reset - resetina visus stilius(istrina stilius).
Mormalize - suvienodina visus stilius tarp narsykliu ir panaikina dali kenkejisku stiliu.

CSS prioritetai

  1. Inline Style - 1000
  2. ID - 100
  3. Class - 10
  4. Element 1
  5. CSS points
  6. CSS Order (in CSS code)
  7. Default styles
  8. Inheritance

GitHub - svetaine, kurioje keliamos kodo versijos

  1. isirasyti Git'a
  2. Patikrinti ar git'as irasytas terminale parasius "git --version"
  3. susikurti github'a paskyra
  4. config'inti git'a
    1. git config --global user.name AusraZio
    2. git config --global user.email ausra.ziogaite@gmail.com
    3. git config --global core.editor code
    4. git config --global core.autocrlf true | input (true jeigu windows)
    5. git config --global -e
  5. patikrinti ar tvarkingas cinfig failas- git config --global --edit
  6. Source Control dalyje paspausti Publish to Github

tagu sarasas

  • p - paragraph - paragrafas skirtas rasyti dideliam kiekiui informacijos
  • h1-h6 - heading1-heading6 - antrastes skirtos pavadinimams ir skiriasi teksto dydziu
  • img - image - paveiksliuka
  • a - hypertext anchor (link) - nuoroda nukelenti kazkur
  • hr - thematic break (horizontal line) - sukuria eilutes luzy su linija
  • br - line break - eilutes luzis
  • ul - unordered list - nerikiuotas sarasas
  • ol - order list - rikiuotas sararasas
  • li - list item - saraso elementas (skirtas rikiuoti arba nerikiuoti sarasiu
    • meta - skirtas nurodyti puslapio meta duomenis
  • title - skirtas nurodyti puslapio tab/o pavdinima

atributu sarasas

  • lang - lanhuage - nurodo kokia kalba tame tag/e (naudinga automatinimas skaitytuvams)
  • charset - character set - simboliu rinkinys nurodo kokia biblioteka naudoti
  • src - source - saltinis nurodantis faila/saltini
  • alt - alternative text - alternatyvus tekstas paveikliuko neuzkrovimo atveju
  • width - nurodo elemento ploti
  • height - nurodo elemento auksti
  • href - hyperlink reference - nuoroda i puslapi arba faila
  • target - nurodo kur atidaryti nuoroda

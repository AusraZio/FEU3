Events

Ivykiai - puslapyje vykstantys kazkokie veiksmai/ivykiai (peles paspaudimas, kliavaturos paspaudimas, informacijos pazymejimas, scroll'inimas, formos pateikimas, failo ikelimas/atsisiuntimas..)
Noredami stebeti ar nutinka koks ivykis, turime pradzioje nurodyti(selektinti) kurioje DOM dalyje tas ivykis gali nutikti.
Nusirodzius (selektinus), kurioje dalyje klausomes ivykio - reikia aprasyti ivykio klausymasi. Tai padarome rasydami "addEventListener" metoda, kuris priima 2 parametrus: koksIvykis (paspaudimas peles, klaviaturos paspaudimas, peles uzejimas), kaDarytiNutikusIvykiui.

Document Object Model

Dokumentas - puslapis, kuriame yra visa vartotojui atvaizduota informacija (tekstas, paveiksliukas, forma, lentele....).
Objektas - kazkoks puslapio elementas (nuo html tag'o, iki ikonos ar span tag'o)
Modelis - objektu struktura kazkuriame elemente (tekstas, paveiksliukas, forma, lentele....).

Noredami per JS selektinti kazkuria DOM dali, turime rasyti toki koda: "document.querySelector" arba "document.querySelectorAll". "QuerySelector" priima 1 parametra - selektoriu (toki koki rasem CSS'e)
